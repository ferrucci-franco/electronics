Observation : Le vecteur rouge montre C×zn. L'angle total est φ + n×ω !

Concept Fondamental

C · zn = A·|z|n · ej(φ + nω)
Point clé : Multiplier un nombre complexe C par zn ajoute nω à la phase de C et multiplie son amplitude par |z|n.

Démonstration mathématique :

Soit C = A·ejφ un nombre complexe quelconque.

Si z = |z|·ejω, alors :

C · zn = A·ejφ · |z|n·ejnω
= A·|z|n · ej(φ + nω)

Application Pratique

Exemple concret :

Si z = ejπ/4 (ω = π/4, rotation de 45°), alors :

  • z1 → rotation de ω = 45°
  • z2 → rotation de 2ω = 90°
  • z4 → rotation de 4ω = 180°
  • z8 → rotation de 8ω = 360° = 0°
Interprétation :

Cette propriété est fondamentale en traitement du signal numérique, notamment pour les transformées de Fourier discrètes où les rotations régulières permettent de décomposer les signaux en fréquences.
